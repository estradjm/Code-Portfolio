# My Custom Vim Configuration

Support for Git version control, directory browsing, status line with glyphs, syntax highlighting, key mappings for running Python with F4 key, searching and PEP8 code standard code highlighting and error checking (linter) for Python, using Terminator for screen management/splitting.
